# Kalkulator Zakat

Aplikasi ini adalah suatu aplikasi penghitung zakat sederhana. Terdapat penghitungan untuk Zakat Fitrah, Zakat Emas, Zakat Perak, Zakat Rikaz, Zakat Pertanian, Zakat Hewan.
